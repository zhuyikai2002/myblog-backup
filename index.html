妥协与进化，结构大改版！
发表于2026-01-10|极客生活
今天是这个音箱项目突飞猛进的一天。本来以为只是修修补补，结果为了适配打印机和优化安装体验，我把整个结构逻辑重盘了一遍。从 280mm 缩减到 250mm 适配打印机，从“瓶中造船”改为“侧门全开”的安装方式，虽然过程有点烧脑，但现在的方案比一开始的“完美”太多了。

发表于2026-01-09|极客生活
今天的时间线被切得碎碎的，感觉过得特别快。下午心血来潮想把音箱的外壳建模搞定，结果掉进了 Blender 的坑里——单位陷阱、布尔运算的幽灵。练车排队等了一个多小时，拿到心心念念的圆形电压表。虽然今天没敲代码，但看着桌上的新配件，还是觉得挺踏实的。

发表于2026-01-08|嵌入式折腾日记

告别自言自语，让 STM32 听懂我的话 (UART中断实战)
发表于2026-01-07|嵌入式折腾日记
之前用 OpenOCD 点亮了呼吸灯，今天决定换个玩法：用串口 (UART) 跟板子聊天。作为 Linux 嵌入式原生玩家，体验了 VS Code 强大的 Tab 补全功能，随后抛弃轮询拥抱中断。使用 USB-TTL 配合 stm32flash 进行 ISP 烧录，在 Linux 终端用 picocom 充当串口助手，踩了“手滑”、括号缺失、引脚配置等坑，最终实现中断回调控制 LED。
把 Arch Linux 装进固态 U 盘，打造随身“练功房”
发表于2026-01-06|折腾记录
手头有一个努米诺固态 U 盘，突发奇想把 Arch Linux 装进去，打造一个完全独立的、随插随用的 x86 开发环境。使用 archinstall 快速安装，配置 Hyprland 平铺窗口管理器，解决断网、中文显示、蓝牙连接等坑，为 STM32 开发做准备，决定强制自己使用 Neovim + GCC + Makefile + OpenOCD 的纯终端流。
