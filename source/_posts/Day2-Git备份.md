## 昨夜：失眠时的基建

故事要从昨天半夜说起。
躺在床上翻来覆去睡不着，既然身体疲惫但大脑清醒，我决定起来做点“基建工作”。
深夜的终端格外安静，我主要干了两件事，把 VPS 的底子打好了：

1.  **配置 SSH 密钥（id_rsa）**：
    为了以后不再每次输入密码，我配置了公钥私钥认证。中间虽然遇到了 `Host key verification failed` 的报错，但排查后顺利解决。现在 `ssh myvps` 秒进服务器，这是后续自动化的基础。
2.  **部署 HTTPS 证书**：
    利用开源项目，我给博客上了 SSL 证书。看着浏览器地址栏里的“不安全”变成了**绿色的小锁**，心里的安全感也随之点亮。

做完这些已是深夜，虽然困，但睡得很踏实。

## 今日：驾校归来的自动化

时间来到今天。
在驾校折腾了一整天练车，晚上八点多才回家。
本想直接休息，但想到昨晚打好的基础，总觉得还差最后一口气。于是我拒绝了娱乐，再次坐到了 Linux Mint 前。

今天的目标很明确：**把重复劳动消灭掉**。

1.  **Git 云端备份**：
    把博客源码推送到了 GitHub 私有仓库。这相当于给我的数字资产买了一份保险，哪怕本地电脑崩溃，代码也能随时复活。
2.  **实现全自动化部署**：
    这是今天的重头戏。我写了一个 `deploy.sh` 脚本，并利用 VS Code 的 Task 系统，把它绑定成了默认构建任务。
    
    现在，不需要手动敲一堆命令，我只需要轻轻按一下 **`Ctrl + Shift + B`**。
    
    清理 -> 生成 -> 上传 -> 备份，所有动作一气呵成。看着终端里滚动的日志，一天的疲惫好像都消散了。

## 黎明前的等待

昨晚搞定了连接与安全，今天搞定了备份与部署。
软环境（Linux + Nginx + HTTPS + Git + 自动化脚本）已经彻底完备。

我看了一眼物流信息。
**明天，我的 STM32 开发套件就要到了。**

阵地已修好，只等核心降临。
明天见，嵌入式世界。
